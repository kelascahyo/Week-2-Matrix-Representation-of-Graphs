# Matrix Representation of Graphs — Crash Course App

An interactive web application built with Streamlit, Python, and NetworkX designed to bridge the intuition gap between visual mathematical graphs and machine-readable data matrices.

## Features
- **Intuitive Visuals:** Learn why standard neural networks struggle with non-Euclidean graphs.
- **Graph to Matrix:** Interactively build connections between nodes and instantly see its Adjacency Matrix & Edge list update.
- **Matrix to Graph:** Directly edit individual matrix indexes to manipulate and draw your own custom node configuration dynamically.
